# **App Name**: Vertex

## Core Features:

- 3D Interactive Hero Scene: An immersive landing section featuring an interactive 3D object using React Three Fiber to showcase high-end technical depth.
- Case Study Intelligence Tool: An AI-powered tool that helps draft technical case studies from raw project descriptions, optimizing the tone for a professional audience.
- Project Showcase Grid: A sophisticated layout featuring large-scale media blocks for high-resolution project renders and development details.
- Fluid Scroll Navigation: Cinematic transitions between portfolio sections that synchronize with 3D model camera movements for a continuous user experience.
- Technical Stack Visualizer: A sleek UI component that displays a modern tech stack with reactive hover states and 3D depth effects.
- Context-Aware Contact Hub: A minimalist, responsive form with floating UI elements that integrate into the global 3D environment.

## Style Guidelines:

- A deep dark color scheme featuring a primary Cobalt Blue (#4D70D6), a desaturated Dark Ink background (#10131A), and a high-contrast Electric Cyan (#4DE0FF) as the accent color.
- A font pairing consisting of 'Space Grotesk' (sans-serif) for high-impact tech headlines and 'Inter' (sans-serif) for readable, neutral body text.
- Ultra-thin SVG outlines with neon glow filters, creating a consistent light-trail effect when hovered.
- A sophisticated asymmetric bento grid that utilizes plenty of white space to prioritize 3D asset visibility and readability.
- Silky smooth 60fps animations leveraging Framer Motion and Lenis scroll for a frictionless, 'organic' interactive feel.
